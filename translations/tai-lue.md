# ᦢᦹᧉᦷᦎᦲ — Dynamic SubAgent

Dynamic SubAgent ᦀᦳᧈᦈᧁᦷᦠᧅ AstrBot ᦢᦲᧃᦈᧅ ᦀᦳᧈᦂᦱᧂᦗᧃ ᦙᦲᧃᦉᦸᧂᦙᦲᧃ ᦀᦳᧈᦑᦲᧃᦉᦻ ᦀᦳᧈᦂᦱᧂᦗᧃ ᦊᦴᧃᦉᦲᧃᦊᦴᧃ។ ᦙᦲᧃᦉᦸᧂᦙᦲᧃ ᦀᦳᧈᦆᦹᧃᦑᦲᧃ ᦀᦳᧈᦊᦱᧃᦉᦸᧃ (safe/medium/full) ᦀᦳᧈᦞᦱᧃ ᦀᦳᧈᦂᦲᧃᦞᦱᧃᦉᦲ, ᦀᦳᧈᦂᦱᧂᦗᧃ ᦊᦴᧃᦉᦲᧃᦊᦴᧃ ᦙᦲᧃᦆᦱᧃ ᦀᦳᧈᦂᦱᧂᦗᧃ ᦊᦴᧃᦉᦲᧃᦊᦴᧃ ᦀᦳᧈᦑᦲᧃᦉᦻ, ᦀᦳᧈᦐᦲᧃᦉᦸᧂᦉᦲᧃ ᦉᦸᧂᦑᦲᧃ ᦀᦳᧈᦞᦱᧃᦉᦲᧃᦞᦱᧃ。

# လိၵ်ႈတႆး (Shan) — Dynamic SubAgent

Dynamic SubAgent ၶေႃႈႁဵတ်း AstrBot ၽလႆႈႁဝ်းၼမ် Agent ၵေႃႈႁၼ် ၸွမ်ႁဵတ်း Sub-Agent ၼမ်ၼေႃႈႁဵတ်း။ ႁဵတ်းၵၼ်ႁူမ်ၼပ်ႉ 3 ၼိုင်ႈ (safe/medium/full) ၸၼ်ႁဵတ်းၵၼ်ႁူမ်ၼပ်ႉႁႃႈႁဵတ်း Sub-Agent ၼမ်ၼေႃႈႁဵတ်း ၸၼ်ႁဵတ်းၸူၼ်ႁဵတ်းႁႃႈႁဵတ်းႁဝ်ႁၼ်ႁဵတ်း။ ယၼ်ႁဵတ်းႁူမ်ၼပ်ႉၼမ်ႁိင်ႁဝ်ႁၼ်ႁဵတ်း။ ႁဵတ်းၵၼ်ႁူမ်ၼပ်ႉၼမ်ႁိင်ႁဝ်ႁၼ်ႁဵတ်းႁဝ်ႁၼ်ႁဵတ်းၼမ်ႁိင်ႁဝ်ႁၼ်ႁဵတ်းၼမ်ႁိင်ႁဝ်ႁၼ်ႁဵတ်း။

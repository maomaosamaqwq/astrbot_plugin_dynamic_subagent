# ภาษาไทยดำ (Thai Dam) — Dynamic SubAgent

Dynamic SubAgent เป็นปลั๊กอิน AstrBot ที่ช่วยให้ Agent หลักสามารถสร้างและจัดการ Sub-Agent ได้อย่างยืดหยุ่น มีการแบ่งแยกสิทธิ์ 3 ระดับ (safe/medium/full) และจำกัดความลึกของการซ้อนกัน Sub-Agent ไม่สามารถสร้าง Sub-Agent ใหม่ได้ รับประกันความปลอดภัยของระบบ รองรับความจำแบบถาวร การมอบหมายงาน และการติดตามการทำงานร่วมกัน

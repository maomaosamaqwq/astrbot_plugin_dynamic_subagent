# ئۇيغۇرچە — Dynamic SubAgent

Dynamic SubAgent بىر AstrBot قىستۇرما بولۇپ، ئاساسىي agent غا ياردەمچى agent لارنى ھەرىكەتچان ھالدا قۇرۇش ۋە باشقۇرۇش ئىمكانىيىتى بېرىدۈ ئۈچ دەرىجىلىك ھوقۇق ئايرىش (safe/medium/full) ۋە چوڭقۇرلۇق چېكى بار، ياردەمچى agent لار باشقا ياردەمچى agent لارنى قۇرالمايدۇ، سىستېما بىخەتەرلىكىگە كاپالەتلىك قىلىدۇ. مۇقىم ئەسلىمە، ۋەزىفە يۆتكەش ۋە ھەمكارلىق ئىز قوغلاشنى قوللايدۇ.

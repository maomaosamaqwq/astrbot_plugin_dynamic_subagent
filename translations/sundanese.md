# Basa Sunda — Dynamic SubAgent

Dynamic SubAgent téh hiji plugin AstrBot anu ngamungkinkeun agen utama nyieun jeung ngatur sub-agen sacara dinamis. Boga tilu tingkat isolasi idin (safe/medium/full) jeung wates jerona, sub-agen teu bisa nyieun sub-agen séjén, ngajamin kaamanan sistem. Ngadukung mémori permanén, mindahkeun tugas jeung ngalacak kolaborasi.

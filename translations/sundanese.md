# Sundanese — Dynamic SubAgent

Dynamic SubAgent mangrupikeun plugin AstrBot anu ngamungkinkeun Agent utama nyieun sareng ngatur subAgent sacara dinamis. Boga tilu tingkat isolasi (safe/medium/full) sareng watesan jero nested, subAgent teu tiasa nyieun subAgent deui, ngajaga kaamanan sistem. Ngadukung memori persisten, mindahkeun tugas, sareng palacakan kolaborasi.

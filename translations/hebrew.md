# עברית (עברית) — Dynamic SubAgent

Dynamic SubAgent הוא תוסף ל-AstrBot שמאפשר לסוכן הראשי ליצור ולנהל תת-סוכנים באופן דינמי. התוסף כולל שלוש רמות בידוד הרשאות (safe/medium/full) ומגבלת עומק קינון. תת-סוכן אינו יכול ליצור תת-סוכנים נוספים, מה שמבטיח את בטיחות המערכת. התוסף תומך בזיכרון מתמשך,העברת משימות ומעקב אחר שיתוף פעולה. התוסף תומך בכלי MCP ובניהול מחזור חיים מלא.

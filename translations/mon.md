# មន — Dynamic SubAgent

Dynamic SubAgent ជា plugin របស់ AstrBot ដែលអនុញ្ញាតឱ្យ Agent មេអាចបង្កើត និងគ្រប់គ្រង Agent កូនបានយ៉ាងថាមវន្ត។ មានកម្រិតអនុញ្ញាតបីកម្រិត (safe/medium/full) និងកម្រិតជម្រៅការរៀបចំ ដែល Agent កូនមិនអាចបង្កើត Agent កូនបានទៀតទេ ដើម្បីធានាសុវត្ថិភាពប្រព័ន្ធ។ គាំទ្រការផ្ទុកអនុស្សាវរីយ៍អចិន្ត្រៃយ៍ ការផ្ទេរភារកិច្ច និងការតាមដានកិច្ចសហប្រតិបត្តិការ។

# Jamamadi — Dynamic SubAgent

Dynamic SubAgent ka AstrBot plugin nha, main Agent pa sub-Agent criar e gerenciar dinamicamente. Tem três níveis de isolamento de permissão (safe/medium/full) e limite de profundidade de aninhamento, sub-Agent não pode criar outros sub-Agentes, garantindo segurança do sistema. Suporta memória persistente, transferência de tarefas e rastreamento de colaboração.

# Mori — Dynamic SubAgent

Dynamic SubAgent sabua plugin AstrBot umba mpowosei Agent utama mpowatu mpo'aturi subAgent dinamik. Mie tolu tingka isolasi (safe/medium/full) mpae batesa kalengge nested, subAgent uma mpowatu subAgent wewo, mpajaga kajari'a sistem. Mpowosei memori persisten, mpapeo' tugas, mpae palacakan kolaborasi.

# ကယား — Dynamic SubAgent

Dynamic SubAgent သည် AstrBot ၏ ပလပ်ဂင်တစ်ခုဖြစ်ပြီး၊ အဓိက Agent ကို သားသမီး Agent များကို အသုံးပြုဖန်တီးပြီး စီမံခန့်ခွဲနိုင်သည်။ ခွင့်ပြုချက်အဆင့် သုံးဆင့် (safe/medium/full) နှင့် nesting အနက်ကန့်သတ်ချက်ရှိပြီး၊ သားသမီး Agent က သားသမီး Agent ထပ်ဖန်တီး၍မရတော့ပါ၊ စနစ်လုံခြုံမှုကို အာမခံသည်။ အမြဲတမ်းမှတ်ဉာဏ်၊ အလုပ်လွှဲပြောင်းခြင်းနှင့် ပူးပေါင်းဆောင်ရွက်မှု ခြေရာခံခြင်းကို ပံ့ပိုးသည်။

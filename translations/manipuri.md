# মৈতৈলোন — Dynamic SubAgent

Dynamic SubAgent অসি AstrBot গী এক প্লাগইননি, মসিনা মূল Agent না অপুম্বা সৃষ্টি অমসুং মপুং থিন্নবা যাওনবা অমা অদুবু অয়াবা যেংনি। অহুম মায়কী লেভেলগী অনুমতি অয়াবা (safe/medium/full) অমসুং নেস্টিংগী অয়েনবা মওং, অপুম্বা Agent না অপুম্বা Agent অনৌবা যান্নবা নত্তবা, সিস্টেমগী অফোবা শক্তম তাওনবা অমা অদুবু। অহুমগী মিনিং, থৌদাম হোংদোকপা অমসুং শন্নবা মতাং থিনবা সহায় তারি।

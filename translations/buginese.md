# Buginese — Dynamic SubAgent

Dynamic SubAgent iyaro plugin AstrBot-nge riasengngi purani Agent utama mappakke subAgent dinamik. Engka tellu tingkat isolasi (safe/medium/full) na bates karoppong nested, subAgent de' gaga mappakke subAgent pole, arajangngi kasalamatan sistem. Napasengngi memori persisten, appallari tugas, na palacakan kolaborasi.

# Kenyah — Dynamic SubAgent

Dynamic SubAgent iko plugin AstrBot mere kakuasaan Agent utama ngaké ngatur subAgent dinamik. Iko telu tingkat isolasi (safe/medium/full) iko batas dalam nested, subAgent kita' ngaké subAgent do', ngajaga selamat sistem. Nyokong memori persisten, mindah tugas, iko palacakan kolaborasi.

# 希腊语 (Greek) — Dynamic SubAgent

Το Dynamic SubAgent είναι ένα πρόσθετο του AstrBot που επιτρέπει στον κύριο Agent να δημιουργεί και να διαχειρίζεται δυναμικά υπο-πράκτορες. Διαθέτει τριεπίπεδη απομόνωση δικαιωμάτων (safe/medium/full) και περιορισμούς βάθους φωλιάσματος — οι υπο-πράκτορες δεν μπορούν να δημιουργήσουν νέους υπο-πράκτορες, εξασφαλίζοντας την ασφάλεια του συστήματος. Υποστηρίζει μόνιμη μνήμη, παράδοση εργασιών και εντοπισμό συνεργασίας.

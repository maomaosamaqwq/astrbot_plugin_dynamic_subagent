# فارسی (فارسی) — Dynamic SubAgent

Dynamic SubAgent یک پلاگین AstrBot است که به عامل اصلی اجازه می‌دهد عامل‌های فرعی را به صورت پویا ایجاد و مدیریت کند. این پلاگین دارای سه سطح مجوز (safe/medium/full) و محدودیت عمق تودرتو است. عامل‌های فرعی نمی‌توانند عامل‌های فرعی دیگری ایجاد کنند، که این امر امنیت سیستم را تضمین می‌کند. از حافظه پایدار، انتقال وظایف و ردیابی همکاری پشتیبانی می‌کند. همچنین ادغام ابزارهای MCP و مدیریت کامل چرخه حیات را پشتیبانی می‌کند.

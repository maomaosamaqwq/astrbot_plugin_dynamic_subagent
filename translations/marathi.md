# मराठी (Marathi) — Dynamic SubAgent

Dynamic SubAgent हा एक AstrBot प्लगइन आहे जो मुख्य Agent ला गतिशीलपणे उप-Agent तयार करण्याची आणि व्यवस्थापित करण्याची सुविधा देतो.

## मुख्य वैशिष्ट्ये

- **तीन-स्तरीय परवानगी वेगळे करणे:**
  - `safe` — मर्यादित परवानग्या, फक्त वाचणे आणि मूलभूत साधने
  - `medium` — विस्तृत परवानग्या, मर्यादित साधन प्रवेश
  - `full` — संपूर्ण परवानग्या, सर्व साधने प्रवेश
- **सुरक्षित वेगळे करणे:** उप-Agent पुन्हा उप-Agent तयार करू शकत नाहीत, सिस्टम सुरक्षा सुनिश्चित
- **कायम स्मृती:** उप-Agent मध्ये ज्ञान सामायिक करणे
- **कार्य हस्तांतरण:** एजंटमध्ये कार्य हस्तांतरण आणि सहकार्य ट्रॅकिंग
- **MCP साधने:** बाह्य साधन जोडणीसाठी MCP प्रोटोकॉल समर्थन
- **जीवनचक्र व्यवस्थापन:** उप-Agent निर्मिती, थांबवणे, पुन्हा सुरू करणे आणि हटवणे

Dynamic SubAgent आपोआप मुख्य Agent कडून येणाऱ्या विनंतींची प्रक्रिया करतो, उप-Agent च्या क्रियाकलापांची नोंद करतो, आणि सिस्टम स्थिरता सुनिश्चित करतो.

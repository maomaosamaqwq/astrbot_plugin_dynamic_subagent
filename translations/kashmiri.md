# कٲشُر (Kashmiri) — Dynamic SubAgent

Dynamic SubAgent اَسٹر بُٹ پلگ اِن چھ، یے مُکھیَل Agent ہیِک سُب ایجنٹ بنأوِتھ تہٕ سامبھالِتھ۔ ترٛؠ لیولَس ہُنَد اِجازَت علیحدگی (safe/medium/full) تہٕ نیسٹنگ سُرحد چھ، سُب ایجنٹ نہٕ ہیِک دویِم سُب ایجنٹ بناونِتھ، یُس سِسٹمَس حفاظَت یَقینی بَناوَن چھ۔ یے پائیدار یاد، کامَس تفویض تہٕ تعاوُنَس پَتہٕ لگاوَنَس مَدد گَژھان چھ۔

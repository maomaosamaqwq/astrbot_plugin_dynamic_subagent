# Burmese — Dynamic SubAgent

Dynamic SubAgent သည် AstrBot ၏ plugin တစ်ခုဖြစ်ပြီး ပင်မ Agent အား sub-Agent များကို ဒိုင်နမစ်ဖြင့် ဖန်တီးစီမံခန့်ခွဲနိုင်စေပါသည်။ အဆင့် ၃ ဆင့် ခွင့်ပြုချက် သီးခြားဖြတ်တောက်ခြင်း (safe/medium/full) နှင့် အတွင်းထည့်နက်ရှိုင်းမှု ကန့်သတ်ချက်ရှိပြီး sub-Agent များသည် sub-Agent အသစ်များ ထပ်ဖန်တီး၍မရပါ၊ စနစ်၏ လုံခြုံရေးကို အာမခံပါသည်။ အမြဲတမ်းမှတ်ဉာဏ်၊ တာဝန်လွှဲပြောင်းခြင်းနှင့် ပူးပေါင်းဆောင်ရွက်မှု ခြေရာခံခြင်းကို ပံ့ပိုးပေးပါသည်။

# 菲律宾语 (Filipino) — Dynamic SubAgent

Ang Dynamic SubAgent ay isang AstrBot plugin na nagbibigay-daan sa pangunahing Agent na lumikha at pamahalaan ang mga sub-Agent nang dinamika. Mayroon itong tatlong antas ng paghihiwalay ng pahintulot (safe/medium/full) at limitasyon sa lalim ng pagka-nested — ang mga sub-Agent ay hindi makakalikha ng sarili nilang sub-Agent, tinitiyak ang seguridad ng sistema. Sinusuportahan ng plugin ang paulit-ulit na memorya, paglilipat ng gawain, at pagsubaybay sa kolaborasyon.

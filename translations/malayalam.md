# മലയാളം (Malayalam) — Dynamic SubAgent

Dynamic SubAgent ഒരു AstrBot പ്ലഗിനാണ്, പ്രധാന Agent-ന് ഉപ-Agent-കളെ ഗതികമായി സൃഷ്ടിക്കാനും നിയന്ത്രിക്കാനും അനുവദിക്കുന്നു.

## സവിശേഷതകൾ

- **ത്രി-തല അനുമതി വേർതിരിവ്**: safe / medium / full — ഉപ-Agent-കളുടെ ശേഷികൾ നിയന്ത്രിക്കുക
- **സുരക്ഷാ പരിധി**: ഉപ-Agent-കൾക്ക് വീണ്ടും ഉപ-Agent-കളെ സൃഷ്ടിക്കാൻ കഴിയില്ല, സിസ്റ്റത്തിന്റെ സുരക്ഷ ഉറപ്പാക്കുന്നു
- **സ്ഥിരമായ ഓർമ്മ**: സെഷനുകൾക്കിടയിൽ അറിവ് നിലനിർത്തുന്നു
- **ടാസ്ക് കൈമാറ്റം**: സങ്കീർണ്ണമായ പ്രശ്നങ്ങൾ ഫലപ്രദമായി ഏകോപിപ്പിക്കുന്നു
- **സഹകരണ ട്രാക്കിംഗ്**: ഉപ-Agent-കളുടെ പ്രവർത്തനങ്ങൾ നിരീക്ഷിക്കുന്നു
- **MCP ടൂളുകൾ**: ബാഹ്യ ഉപകരണങ്ങളുമായി സംയോജനം
- **ലൈഫ് സൈക്കിൾ മാനേജ്മെന്റ്**: ഉപ-Agent-കളുടെ പൂർണ്ണ നിയന്ത്രണം

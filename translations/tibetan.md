# བོད་ཡིག — Dynamic SubAgent

Dynamic SubAgent ནི་ AstrBot ཨེབ་ཆས་ཤིག་ཡིན་པ་དང་། གཙོ་བོའི་ Agent ལ་ Sub-Agent འཕྲུལ་ཆས་ཀྱིས་བཟོ་བ་དང་དོ་དམ་བྱེད་པར་རོགས་རམ་བྱེད། འགྲོ་ལུགས་རིམ་པ་གསུམ་གྱི་ཁེ་དབང་ལོགས་དབྱེ་ (safe/medium/full) དང་གཏིང་ཚད་ཚད་བཀག་ཡོད་པས། Sub-Agent གཞན་བཟོ་མི་ཐུབ་པས་རྒྱུད་ཁོངས་ཀྱི་བདེ་འཇགས་འགན་ལེན་བྱེད། རྒྱུན་གནས་དྲན་ཤེས་དང་། ལས་ཀ་སྤོ་སྒྱུར། མཉམ་ལས་རྗེས་འདེད་བཅས་ལ་རྒྱབ་སྐྱོར་བྱེད།

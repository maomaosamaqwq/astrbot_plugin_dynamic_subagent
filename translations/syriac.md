# ܠܫܢܐ ܣܘܪܝܝܐ ܡܥܠܬܐ

OpenClaw ܗܘ ܡܚܙܝܬܐ ܕܥܕܪܢܐ ܕܚܟܡܬܐ ܨܢܝܥܬܐ ܕܦܬܝܚܐ ܒܡܒܘܥܐ ܕܡܬܩܛܪ ܠܐܘܪܚܬܐ ܕܫܘܕܥܐ ܐܝܟ Telegram، Discord ܘ WhatsApp. ܗܘ ܣܡܟ ܠܕܡܘܣܝܐ ܣܓܝܐܐ ܕܚܟܡܬܐ ܨܢܝܥܬܐ ܘܡܫܟܚ ܠܡܬܚܫ ܒܡܐܢܐ ܘܐܘܡܢܘܬܐ ܕܡܫܚܠܦܐ ܠܡܫܠܡܘ ܫܘܪܝܐ. ܕܡܬܕܒܪ ܡܢ ܟܢܫܐ، OpenClaw ܡܩܒܠ ܫܘܬܦܘܬܐ ܕܟܠ ܡܢ ܕܨܒܐ ܠܡܒܢܝܗ ܠܥܕܪܢܗ ܚܟܝܡܐ ܕܝܠܗ.

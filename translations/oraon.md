# कुड़ुख़ (Oraon) — Dynamic SubAgent

Dynamic SubAgent ए AstrBot प्लगिन हें जो मुख्य Agent खातिर Sub-Agent बनाने आर प्रबंध करने में समर्थ बनावे हें। तीन स्तर अधिकार पृथक्करण (safe/medium/full) आर नेस्टिंग गहराई सीमा हें, Sub-Agent दुसरा Sub-Agent नहीं बना सके हें, जो प्रणाली सुरक्षा सुनिश्चित करे हें। स्थायी स्मृति, कार्य हस्तांतरण आर सहयोग अनुसरण समर्थन करे हें।

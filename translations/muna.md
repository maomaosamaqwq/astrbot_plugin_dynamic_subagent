# Muna — Dynamic SubAgent

Dynamic SubAgent noanoa plugin AstrBot umba nopohawu Agent utama mpobhahi nopo'atu subAgent dinamik. Nomie tolu tingka isolasi (safe/medium/full) nomie bhatasa kalengka nested, subAgent te'atu mpobhahi subAgent bhe, nopojaga kanoanoa sistem. Nopohawu memori persisten, nopalele tugas, nomie palacakan kolaborasi.

# Kannada — Dynamic SubAgent

Dynamic SubAgent ಒಂದು AstrBot ಪ್ಲಗಿನ್ ಆಗಿದ್ದು, ಮುಖ್ಯ Agent ಗೆ ಉಪ-ಏಜೆಂಟ್‌ಗಳನ್ನು ಡೈನಾಮಿಕ್ ಆಗಿ ರಚಿಸಲು ಮತ್ತು ನಿರ್ವಹಿಸಲು ಅನುಮತಿಸುತ್ತದೆ. ಮೂರು ಮಟ್ಟದ ಅನುಮತಿ ಐಸೋಲೇಶನ್ (safe/medium/full) ಮತ್ತು ನೆಸ್ಟಿಂಗ್ ಆಳ ಮಿತಿಯನ್ನು ಹೊಂದಿದ್ದು, ಉಪ-ಏಜೆಂಟ್‌ಗಳು ಇನ್ನೊಂದು ಉಪ-ಏಜೆಂಟ್‌ಗಳನ್ನು ರಚಿಸಲು ಸಾಧ್ಯವಾಗುವುದಿಲ್ಲ, ಇದು ಸಿಸ್ಟಮ್ ಭದ್ರತೆಯನ್ನು ಖಾತ್ರಿಪಡಿಸುತ್ತದೆ. ಶಾಶ್ವತ ಸ್ಮರಣೆ, ಕಾರ್ಯ ವರ್ಗಾವಣೆ ಮತ್ತು ಸಹಕಾರ ಟ್ರ್ಯಾಕಿಂಗ್ ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

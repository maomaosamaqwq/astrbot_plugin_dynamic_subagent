# Dynamic SubAgent નું પરિચય

Dynamic SubAgent એ એક AstrBot પ્લગઇન છે જે મુખ્ય Agent ને પેટા Agent ને ગતિશીલ રીતે બનાવવા અને સંચાલિત કરવાની ક્ષમતા આપે છે. આ પ્લગઇન ત્રણ સ્તરની પરવાનગી અલગાવ (safe/medium/full) અને નેસ્ટિંગ ઊંડાઈ મર્યાદા સાથે આવે છે, જેથી પેટા Agent વધુ પેટા Agent બનાવી શકતા નથી, જે સિસ્ટમની સુરક્ષા સુનિશ્ચિત કરે છે. તે કાયમી મેમરી, કાર્ય હસ્તાંતરણ અને સહયોગ ટ્રેકિંગને પણ સપોર્ટ કરે છે.

## વિશેષતાઓ

- **ગતિશીલ પેટા Agent બનાવવું**: મુખ્ય Agent જરૂર મુજબ નવા પેટા Agent બનાવી શકે છે
- **ત્રણ સ્તરની પરવાનગી અલગાવ**: safe, medium અને full એમ ત્રણ સ્તરો ઉપલબ્ધ છે
- **નેસ્ટિંગ ઊંડાઈ નિયંત્રણ**: પેટા Agent વધુ પેટા Agent બનાવી શકતા નથી
- **કાયમી મેમરી**: પેટા Agent ની સ્થિતિ અને ડેટા જાળવી રાખે છે
- **કાર્ય હસ્તાંતરણ**: Agent વચ્ચે કાર્યો સરળતાથી સ્થાનાંતરિત કરી શકાય છે
- **સહયોગ ટ્રેકિંગ**: બધી સહયોગ પ્રવૃત્તિઓનું લોગિંગ અને ટ્રેકિંગ

## ઉપયોગનું ઉદાહરણ

```
# મુખ્ય Agent એક પેટા Agent બનાવે છે
create_subagent(task="સંશોધન કાર્ય", permission="safe")

# પેટા Agent કાર્ય પૂર્ણ કરે છે
subagent.complete(result="સંશોધન પૂર્ણ")

# મુખ્ય Agent પરિણામ પ્રાપ્ત કરે છે
get_subagent_result(agent_id="subagent_001")
```

## ઉપયોગી પરિસ્થિતિઓ

- જટિલ કાર્યોને નાના ભાગોમાં વિભાજિત કરવા
- સમાંતર કાર્ય નિષ્પાદન
- સુરક્ષિત વાતાવરણમાં અલગ-અલગ કાર્યો ચલાવવા
- સ્કેલેબલ AI સિસ્ટમ બનાવવા

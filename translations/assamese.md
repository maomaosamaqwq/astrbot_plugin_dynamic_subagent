# অসমীয়া — Dynamic SubAgent

Dynamic SubAgent এটা AstrBot প্লাগিন, যিয়ে মূল Agent-ক সৃষ্টি আৰু পৰিচালনা কৰিবলৈ সক্ষম কৰে। তিনিটা স্তৰৰ অনুমতি বিচ্ছেদ (safe/medium/full) আৰু নেস্টিং গভীৰতা সীমা আছে, সন্তান Agent-এ পুনৰ সন্তান Agent সৃষ্টি কৰিব নোৱাৰে, যি প্ৰণালীৰ সুৰক্ষা নিশ্চিত কৰে। স্থায়ী স্মৃতি, কাৰ্য্য হস্তান্তৰ আৰু সহযোগিতা ট্ৰেকিং সমৰ্থন কৰে।

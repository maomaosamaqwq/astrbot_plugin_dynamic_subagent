# Lao — Dynamic SubAgent

Dynamic SubAgent ແມ່ນ plugin AstrBot ທີ່ອະນຸຍາດໃຫ້ Agent ຫຼັກສາມາດສ້າງ ແລະ ຈັດການ sub-Agent ໄດ້ຢ່າງເປັນປົກກະຕິ. ມີສາມລະດັບການແຍກສິດ (safe/medium/full) ແລະ ຂີດຈຳກັດຄວາມເລິກຂອງການຊ້ອນ, ບ່ອນທີ່ sub-Agent ບໍ່ສາມາດສ້າງ sub-Agent ອື່ນໄດ້, ຮັບປະກັນຄວາມປອດໄພຂອງລະບົບ. ສະໜັບສະໜູນຄວາມຈຳຖາວອນ, ການໂອນຍ້າຍໜ້າທີ່ ແລະ ການຕິດຕາມການຮ່ວມມື.

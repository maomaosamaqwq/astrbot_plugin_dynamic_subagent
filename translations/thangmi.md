# 坦古特语 — Dynamic SubAgent

Dynamic SubAgent ho AstrBot ko ek plug-in ho, jasle mukhya Agent la gatigatama banāune ra byabasthit garne sahakarta dincha. Tin star ko anumati vibhajan (safe/medium/full) ra gahiro simā ho, sahakarta Agent le pheri sahakarta Agent banauna sakdaina, pranali surakshit rakhdcha. Sthayi smriti, karyantaran ra sahyog anusandhan samarthan garcha.

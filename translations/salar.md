# 撒拉语 — Dynamic SubAgent

Dynamic SubAgent AstrBot-ning bir plasini, asıl Agent-ğa balalar Agent-larnı yasap idare etmek imkanı beredi. Üç darajalı hoquq izolatsiyası (safe/medium/full) we nesterlik chetlewsı balalar Agent-lar yene balalar Agent yasalmaydı, sistema xawepsizligini teminlaydı. Daimiy xotire, wezipelarni tapsıru we hemkarlıqni quwalastırışnı desteklaydı.

# Kyrgyz — Dynamic SubAgent

Dynamic SubAgent — бул AstrBot плагини, негизги Agent'ке суб-Agentтерди динамикалык түрдө түзүүгө жана башкарууга мүмкүндүк берет. Үч деңгээлдүү уруксат изоляциясы (safe/medium/full) жана ичине кирүү тереңдигинин чеги бар, суб-Agentтер башка суб-Agentтерди түзө алышпайт, тутумдун коопсуздугун камсыз кылат. Туруктуу эс, тапшырмаларды берүү жана кызматташтыкты көзөмөлдөөнү колдойт.

# 巴鲁语 — Dynamic SubAgent

Dynamic SubAgent merupakan plugin AstrBot yang membolehkan Agent utama mencipta dan menguruskan anak Agent secara dinamik. Mempunyai tiga tahap pengasingan keizinan (safe/medium/full) dan had kedalaman bersarang, anak Agent tidak boleh mencipta anak Agent lagi, memastikan keselamatan sistem. Menyokong penyimpanan memori kekal, penyerahan tugas dan penjejakan kerjasama.

---
*This introduction is written in 巴鲁语 (Baru).*

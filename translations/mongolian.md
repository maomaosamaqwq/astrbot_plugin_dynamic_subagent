# Монгол (Монгол хэл) — Dynamic SubAgent

Dynamic SubAgent нь AstrBot-ийн нэмэлт залгаас бөгөөд Үндсэн Агентэд дэд Агентуудыг динамикаар үүсгэх, удирдах боломжийг олгодог. Гурван түвшний зөвшөөрлийн тусгаарлалт (safe/medium/full) болон гүнзгийрүүлэх хязгаартай. Дэд Агентууд дахин Агент үүсгэх боломжгүй тул системийн аюулгүй байдлыг хангадаг. Тогтмол санах ой, даалгалт шилжүүлэх, хамтын ажиллагааг хянах боломжийг дэмждэг. Мөн MCP багажуудыг нэгтгэх болон бүрэн амьдралын мөчлөгийн удирдлагыг дэмждэг.

# Shona — Dynamic SubAgent

Dynamic SubAgent ndeye AstrBot plugin inobvumira agent mukuru kuti agadzire nekutonga sub-agents nenzira inoshanduka. Ine matatu matanho ekuparadzanisa mvumo (safe/medium/full) nemugero wekudzika, sub-agents havagone kugadzira mamwe sub-agents, ichivimbisa kuchengetedzeka kwe system. Inotsigira ndangariro inogara, kutamisa mabasa nekutevera kubatana.

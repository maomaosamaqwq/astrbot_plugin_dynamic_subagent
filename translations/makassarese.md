# Makassarese — Dynamic SubAgent

Dynamic SubAgent iami plugin AstrBot-nga appa're'nna Agent utama appakke subAgent dinamik. Nia' tallu tingkat isolasi (safe/medium/full) siagang batesa kalumbarra'nna nested, subAgent tena nappakke subAgent lom, anjagai kasalamatan sistem. Appaisengi memori persisten, appassele' tugas, siagang palacakan kolaborasi.

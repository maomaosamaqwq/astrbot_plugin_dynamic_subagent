# অসমীয়া (Ahom) — Dynamic SubAgent

Dynamic SubAgent হৈছে এটা AstrBot প্লাগিন যিয়ে মূল Agent ক স্বতঃত্বভাৱে Sub-Agent সৃষ্টি আৰু ব্যৱস্থাপনা কৰিবলৈ সক্ষম কৰে। তিনিটা স্তৰৰ অনুমতি পৃথকীকৰণ (safe/medium/full) আৰু নেষ্টিং গভীৰতাৰ সীমা আছে, Sub-Agent পুনৰ Sub-Agent সৃষ্টি কৰিব নোৱাৰে, যিয়ে প্ৰণালীৰ সুৰক্ষা নিশ্চিত কৰে। স্থায়ী স্মৃতি, কাৰ্য্য হস্তান্তৰ আৰু সহযোগিতা অনুসৰণৰ সমৰ্থন কৰে।

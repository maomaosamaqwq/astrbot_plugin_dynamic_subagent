# 裕固语 — Dynamic SubAgent

Dynamic SubAgent ne AstrBot-ning bir plugin-i, bu asıl Agent-ge balalar Agent-larnı yasap idare etmek imkanı beredi. Üç darajalı hoquq izolatsiyası (safe/medium/full) we kenikterlik chetlewsı bar, balalar Agent-lar yene balalar Agent yasalmaydı, bu sistema xawepsizligini teminlaydı. Daimiy xotire, wezipelarni tapsıru we hemkarlıqni quwalaştırma destekleydi.

---
*This introduction is written in 裕固语 (Yugur).*

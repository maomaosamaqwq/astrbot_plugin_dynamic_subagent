# 莫多克语 — Dynamic SubAgent

Dynamic SubAgent ke an AstrBot plugin, hecet main Agent podo nupakadun kinchadi kashshush anak Agent nipadu. Nopusunuktu tuptun otoritas isolasi (safe/medium/full) ka nopusunuktu kedalaman nesting, anak Agent nushi nupakadun kinchadi anak Agent balap, nopusunuktu kaamanan sistem. Nyamut penyimpanan memori permanen, penyerahan tugas ka pelacakan kerja sama.

---
*This introduction is written in 莫多克语 (Modok).*

# Minangkabau — Dynamic SubAgent

Dynamic SubAgent marupokan plugin AstrBot nan mamungkinan Agent utamo mambuek jo mangarola subAgent sacaro dinamis. Mampunyoi tigo tingkek isolasi (safe/medium/full) jo bateh kadalaman nested, subAgent indak dapek mambuek subAgent lai, manjago kaamanan sistem. Mandukuang memori persisten, panyalahan tugas, jo palacakan kolaborasi.

# ภาษาไทแดง (Tai Daeng) — Dynamic SubAgent

Dynamic SubAgent เป็นปลั๊กอิน AstrBot ที่ช่วยให้ Agent หลักสร้างและจัดการ Sub-Agent ได้อย่างคล่องตัว มีระบบสิทธิ์ 3 ระดับ (safe/medium/full) และจำกัดความลึกของการซ้อน Sub-Agent ไม่สามารถสร้าง Sub-Agent ใหม่ได้ ทำให้ระบบมีความปลอดภัย รองรับความจำแบบถาวร การมอบหมายงาน และการติดตามความร่วมมือ

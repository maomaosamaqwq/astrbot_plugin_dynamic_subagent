# Achuar — Dynamic SubAgent

Dynamic SubAgent nui ka plugin AstrBot amawisha aentsar aintsar aentsar nii ainia. Nui aentsar nii mampu mencipta dan mengelola sub-agent secara dinamis. Memiliki tiga tingkat isolasi izin (safe/medium/full) dan batasan kedalaman bersarang, sub-agent tidak dapat membuat sub-agent lagi, memastikan keamanan sistem. Mendukung memori persisten, penyerahan tugas, dan pelacakan kolaborasi.

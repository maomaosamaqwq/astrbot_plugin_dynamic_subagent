# बड़ो — Dynamic SubAgent

Dynamic SubAgent बिदां AstrBot नि सोमों प्लागिन, बेयो मुख्य Agent खां साबा Agent नि सोरजों आरो मिगिरि खालामनो हायो। मोनसे मोनथाम इजाजत रोखोमनाय (safe/medium/full) आरो नेस्टिं गोजौनि सिमा दं, साबा Agent आ साबा Agent सोरजों नाङाम, बेयो सिस्टेमनि रैखाथा सुनियोगोनां। गोमथाय मेमरी, खामानि हस्तांतरण आरो सहयोग ट्रेकिंग हेफाजाब हागोन।

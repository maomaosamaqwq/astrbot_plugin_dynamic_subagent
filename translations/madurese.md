# Madurese — Dynamic SubAgent

Dynamic SubAgent panèka plugin AstrBot sè ngamarèken Agent otama nyabâb aghâbây subAgent sacara dinamis. Bâḍâ tello' tingkatan isolasi (safe/medium/full) bân bâtesan ḍâlâm nested, subAgent ta' bisa aghâbây subAgent pole, ngajhâgâ ka'amanan sistem. Nḍukung memori persistèn, alèntar tugas, bân palacakan kolaborasi.

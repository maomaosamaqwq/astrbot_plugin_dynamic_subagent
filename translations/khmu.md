# ຄົມມຸ (Khmu) — Dynamic SubAgent

Dynamic SubAgent ເປັນແພລກອິນ AstrBot ທີ່ອະນຸຍາດໃຫ້ Agent ຫຼັກສ້າງ ແລະ ຈັດການ Sub-Agent ໄດ້ຢ່າງຍືດຫຍຸ່ນ. ມີການແຍກສິດທິ 3 ຂັ້ນ (safe/medium/full) ແລະ ຈຳກັດຄວາມເລິກຂອງການຊ້ອນກັນ, Sub-Agent ບໍ່ສາມາດສ້າງ Sub-Agent ອື່ນໄດ້, ຮັບປະກັນຄວາມປອດໄພຂອງລະບົບ. ສະໜັບສະໜູນການຈົດຈຳແບບຍືນຍົງ, ການໂອນຍ້າຍງານ ແລະ ການຕິດຕາມການຮ່ວມມື.

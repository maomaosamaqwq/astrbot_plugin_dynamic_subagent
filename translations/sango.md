# Sango — Dynamic SubAgent

Dynamic SubAgent zo kété AstrBot na, mo nzöné agent principal té sub-agents mbi é créér mo é gézér na mo tî. Ala nzöné tî permission (safe/medium/full) mo limité tî profondeur, sub-agents é béka créér sub-agents tî ôtré, na défendré sûreté tî système. Mémé persistente, transfert tî tâches mo suivi tî collaboration na mo sörö.

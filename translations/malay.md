# Melayu (Bahasa Melayu) — Dynamic SubAgent

Dynamic SubAgent ialah plugin AstrBot yang membolehkan Agens Utama mencuba dan menguruskan Agens Secara Dinamik. Plugin ini dilengkapi dengan tiga tahap isolasi izin (safe/medium/full) dan had kedalaman bersarang. Agens sekunder tidak boleh mencipta agens baharu, sekali gus memastikan keselamatan sistem. Ia menyokong memori berterusan, peralihan tugas, dan penjejakan kolaborasi. Selain itu, plugin ini turut menyokong integrasi alat MCP serta pengurusan kitaran hayat yang lengkap.

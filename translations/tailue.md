# ᦟᦳᧂᦑᦳᧂᦈᦹᧃ (Tai Lue) — Dynamic SubAgent

Dynamic SubAgent ᦔᦲᧃᦈᦸᧂᦉᦴᧃ AstrBot ᦀᦸᧉᦟᦹᧉᦑᦸᧂ Agent ᦀᦾᧉᦉᦵᦋᦲᧂᧈ ᦀᦸᧉᦓᦲᧃᦙᦻ Sub-Agent ᦓᦵᦌᦲᧃᧉ ᦡᦲᧃᧉ ᦔᦸᧃᧉ ᦈᦹᧃᧈ ᦀᦸᧉᦕᦱᧁᧉ ᦢᦱᧃᧉ ᦀᦸᧉᦊᦻᧉ ᦅᦸᧂᦊᦾᧉ ᦓᦵᦌᦲᧃᧉ ᦕᦸᧃᧉ ᦐᦸᧈ ᦕᦱᧂ ᦀᦾᧉ ᦕᦱᧉ ᦀᦸᧉᦊᦻᧉ ᦅᦸᧂᧈ ᦀᦸᧉᦉᦢᦱᧃᧉ ᦑᦸᧃᧉ ᦀᦸᧉᦀᦳᧂᦊᦸᧈ ᦀᦸᧉᦈᦵᦑᦲᧂᧈ Sub-Agent ᦓᦵᦌᦲᧃᧉ ᦀᦸᧉᦉᦵᦋᦲᧂᧈ Sub-Agent ᦕᦱᧉ ᦓᦵᦌᦲᧃᧉ ᦀᦸᧉᦕᦱᧁᧉ ᦢᦱᧃᧉ ᦀᦸᧉᦊᦻᧉ ᦓᦸᧉᦊᦾᧉ ᦓᦵᦌᦲᧃᧉ ᦀᦸᧉᦊᦻᧉ ᦓᦸᧉᦊᦾᧉ ᦓᦵᦌᦲᧃᧉ ᦀᦸᧉᦊᦻᧉ ᦓᦸᧉᦊᦾᧉ ᦓᦵᦌᦲᧃᧉ ᦀᦸᧉᦊᦻᧉ ᦓᦸᧉᦊᦾᧉ

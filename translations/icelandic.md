# Íslenska (Íslenska) — Dynamic SubAgent

Dynamic SubAgent er viðbót fyrir AstrBot sem gerir aðal-Agent kleift að búa til og stjórna undir-Agentum á skilvirkan hátt. Viðbótin hefur þriggja stiga leyfiseinangrun (safe/medium/full) og takmörkun á hreiðrunardýpi. Undir-Agent getur ekki búið til fleiri undir-Agenta, sem tryggir öryggi kerfisins. Styður varanlegt minni, verkefnayfirfærslu og samstarfsrakningu. Styður MCP verkfæri og fulla lífsferilsstjórnun.

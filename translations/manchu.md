# ᠮᠠᠨᠵᡠ (ᠮᠠᠨᠵᡠ ᡤᡳᠰᡠᠨ) — Dynamic SubAgent

Dynamic SubAgent ᡩᡝᡵᡳ AstrBot ᡳ ᠵᡠᠸᡝ ᠰᡳᡵᡝ ᠪᡳᡨᡥᡝ。 ᡩᡠᡵᡠᠮᡝ ᠪᠠ ᠠᠮᠪᠠ ᠠᡤᡝᡨᡝ ᡩᡝᡵᡳ ᠵᡠᠸᡝ ᠰᡳᡵᡝ ᠪᡳᡨᡥᡝ ᠪᡝ ᠰᡳᡵᡝ ᡩᡝᡵᡳ ᠠᡩᠠᠯᡳ ᡩᡠᡵᡠᠮᡝ ᠪᠠᡥᠠᠮᠪᡳ。 ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝ ᡩᡝᡵᡳ ᡩᡠᡵᡠᠮᡝ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ。 ᠰᡳᡵᡝ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡠᡵᡠᠮᡝ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ。 ᡩᡠᡵᡠᠮᡝ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡠᡵᡠᠮᡝ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ ᡩᡝᡵᡳ ᡥᡡᠸᠠᠯᡳᠶᠠᠮᡝᡷᡳ。

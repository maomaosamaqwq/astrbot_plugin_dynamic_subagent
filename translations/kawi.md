# Kawi — Dynamic SubAgent

Dynamic SubAgent inggih punika plugin AstrBot ingkang ngidini Agent utami kanggé nggawe lan ngatur sub-Agent kanthi dinamis. Gadhi tataran telu isolasi idin (safe/medium/full) winatesan kedalaman nesting, ingkang sub-Agent boten saged nggawe sub-Agent sanesipun, njamin keamanan sistem. Nyengkuyung memori langgeng, pindhah tugas lan lacak kerjasama.

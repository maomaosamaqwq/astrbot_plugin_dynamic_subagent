# Magyar (Magyar) — Dynamic SubAgent

A Dynamic SubAgent egy AstrBot bővítmény, amely lehetővé teszi a fő Agent számára, hogy dinamikusan hozzon létre és kezeljen sub-Agent-eket. A bővítmény három szintű jogosultsági izolációt (safe/medium/full) és beágyazási mélység korlátot biztosít. A sub-Agent nem hozhat létre további sub-Agent-eket, ami garantálja a rendszer biztonságát. Támogatja a tartós memóriát, a feladatátadást és az együttműködés nyomon követését. Támogatja az MCP eszközöket és a teljes életciklus-kezelést.

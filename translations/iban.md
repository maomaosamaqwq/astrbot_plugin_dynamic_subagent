# Iban — Dynamic SubAgent

Dynamic SubAgent tu plugin AstrBot ti meri kuasa Agent utama bebas ngaga lalu ngatur subAgent dinamik. Bisi tiga tingkat isolasi (safe/medium/full) enggau batas dalam nested, subAgent enda ulih ngaga subAgent agi, ngaga aman sistem. Nyukung memori persisten, mindah pengawa, enggau penuntun kolaborasi.

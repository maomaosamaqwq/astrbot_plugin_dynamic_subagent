# Dynamic SubAgent का परिचय

Dynamic SubAgent एक AstrBot प्लगइन है जो मुख्य Agent को उप-Agent को गतिशील रूप से बनाने और प्रबंधित करने की क्षमता प्रदान करता है। यह प्लगइन तीन स्तरीय अनुमति अलगाव (safe/medium/full) और नेस्टिंग गहराई सीमा के साथ आता है, जिससे उप-Agent आगे उप-Agent नहीं बना सकते, जो सिस्टम की सुरक्षा सुनिश्चित करता है। यह स्थायी मेमोरी, कार्य हस्तांतरण और सहयोग ट्रैकिंग को भी सपोर्ट करता है।

## विशेषताएं

- **गतिशील उप-Agent निर्माण**: मुख्य Agent आवश्यकतानुसार नए उप-Agent बना सकता है
- **तीन स्तरीय अनुमति अलगाव**: safe, medium और full तीन स्तर उपलब्ध हैं
- **नेस्टिंग गहराई नियंत्रण**: उप-Agent आगे उप-Agent नहीं बना सकते
- **स्थायी मेमोरी**: उप-Agent की स्थिति और डेटा बनाए रखता है
- **कार्य हस्तांतरण**: Agent के बीच कार्यों को आसानी से स्थानांतरित किया जा सकता है
- **सहयोग ट्रैकिंग**: सभी सहयोग गतिविधियों का लॉगिंग और ट्रैकिंग

## उपयोग का उदाहरण

```
# मुख्य Agent एक उप-Agent बनाता है
create_subagent(task="अनुसंधान कार्य", permission="safe")

# उप-Agent कार्य पूरा करता है
subagent.complete(result="अनुसंधान पूर्ण")

# मुख्य Agent परिणाम प्राप्त करता है
get_subagent_result(agent_id="subagent_001")
```

## उपयोगी परिस्थितियां

- जटिल कार्यों को छोटे भागों में विभाजित करने के लिए
- समानांतर कार्य निष्पादन के लिए
- सुरक्षित वातावरण में अलग-अलग कार्य चलाने के लिए
- स्केलेबल AI सिस्टम बनाने के लिए

# ককবরক — Dynamic SubAgent

Dynamic SubAgent হিদু AstrBot নি গমাই প্লাগিন, বেয়ে মুখ্য Agent খাংসু সাবা Agent নি সৃজনা আরু ব্যাবস্থাপনা গারনো হায়ো। মনসে মনস্তৰী অনুমতি পৃথকীকৰণ (safe/medium/full) আরু নেস্টিং গভীরতা সীমা দং, সাবা Agent আ সাবা Agent সৃজনা নাঙাম, বেয়ে পদ্ধতি নি সুৰক্ষা সুনিশ্চিত গারো। স্থায়ী স্মৃতি, কাৰ্য হস্তান্তৰ আরু সহযোগিতা ট্রেকিং সমৰ্থন গারো।

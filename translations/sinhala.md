# සිංහල — Dynamic SubAgent

Dynamic SubAgent යනු AstrBot ප්ලගිනයකි, ප්‍රධාන ඒජන්ටට උප ඒජන්ටවරුන් ගතිකව සෑදීමට හා කළමනාකරණය කිරීමට ඉඩ දෙයි. මට්ටම් තුනක අවසර වෙන්කිරීමක් (safe/medium/full) හා ගැඹුරු සීමාවක් ඇත, උප ඒජන්ටවරුන්ට තවත් උප ඒජන්ටවරුන් සෑදිය නොහැක, පද්ධතියේ ආරක්ෂාව සහතික කරයි. ස්ථිර මතකය, කාර්ය මාරුව හා සහයෝගීතා පසුවිපරමට සහාය වේ.

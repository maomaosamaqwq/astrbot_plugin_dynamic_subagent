# ภาษาไทย — Dynamic SubAgent

Dynamic SubAgent เป็นปลั๊กอินของ AstrBot ที่ช่วยให้ Agent หลักสามารถสร้างและจัดการ Sub-Agent ได้อย่าง动态 มีการแยกสิทธิ์ 3 ระดับ (safe/medium/full) และจำกัดความลึกของการซ้อนกัน ทำให้ Sub-Agent ไม่สามารถสร้าง Sub-Agent เพิ่มเติมได้ รับประกันความปลอดภัยของระบบ รองรับหน่วยความจำถาวร การโอนย้ายงาน และการติดตามความร่วมมือ

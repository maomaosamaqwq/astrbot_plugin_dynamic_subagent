# Aymara — Dynamic SubAgent

Dynamic SubAgent ukax AstrBot plugin ukawa, maynitayi principal Agent ukax sub-agentes ukanakaxa dinamicamente crear ukhamarakampi gestionar sartana. Nivel kimsa aislamiento permiso (safe/medium/full) ukhamarakampi limite profundidad anidamiento ukax jikxatawiwa, sub-agentes nayrantayirinaka sub-agentes crear jan walt'äkiti, sistema seguridad ukax jikxatawiwa. Memoria persistente, transferencia tareas ukhamarakampi seguimiento colaboración ukax soporta sna.

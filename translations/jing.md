# 京语 — Dynamic SubAgent

Dynamic SubAgent la mot plugin AstrBot, cho phep Agent chinh tao va quan ly Agent con mot cach dong. Co ba cap do phan quyen cach ly (safe/medium/full) va gioi han do sau long nhau, Agent con khong the tao them Agent con, dam bao an toan he thong. Ho tro nho lau dai, ban giao nhiem vu va theo doi hop tac.

# 布拉湾语 — Dynamic SubAgent

Dynamic SubAgent ngemuh AstrBot plug-in, ih mamatai Agent bisi nyaman aguh saba ngagip menurus aguh anak Agent. Tusu level otoritas isolasi (safe/medium/full) ukap pembatasan kedalaman nesting, anak Agent tansi bisi nyaman aguh saba anak Agent balap, mamatai kaamanan sistem. Nyamut penyimpanan memori permanen, penyerahan tugas ukap pelacakan kerja sama.

---
*This introduction is written in 布拉湾语 (Berawan).*

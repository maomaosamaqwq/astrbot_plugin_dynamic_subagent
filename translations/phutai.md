# ภาษาภูไท (Phutai) — Dynamic SubAgent

Dynamic SubAgent เป็นปลั๊กอิน AstrBot ที่ให้ Agent หลักสร้างและจัดการ Sub-Agent ได้อย่างยืดหยุ่น มีการแยกสิทธิ์ 3 ระดับ (safe/medium/full) และจำกัดความลึกของการซ้อนกัน Sub-Agent ไม่สามารถสร้าง Sub-Agent ใหม่ได้ รับประกันความปลอดภัยของระบบ รองรับความจำถาวร การโอนงาน และการติดตามการทำงานร่วมกัน

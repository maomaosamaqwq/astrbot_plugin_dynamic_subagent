🧬 子 Agent Dynamic SubAgent

AI 通过 tool call 动态创建和管理的子 Agent 概览

🤖 子 Agent 由主 AI 通过以下 LLM Tool 管理：

  • spawn_agent(name, description, instruction, task?, permission_level?) — 创建子 Agent
  • list_agents() — 列出所有活跃子 Agent
  • delete_agent(name) — 销毁子 Agent
  • transfer_to_<name>(task) — 将任务转交给指定子 Agent

💡 建议传 task 参数给 spawn_agent，创建后立即执行。
